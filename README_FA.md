# سامانه کارگاه تراشکاری — Attendance Calendar UX

این نسخه شامل اصلاحات تجربه کاربری حضور و غیاب است:

- منوی جدید «حضور و غیاب» در پنل ادمین
- نمای تقویمی ماهانه شبیه Google Calendar / Google Tasks
- نمایش حضور، خروج موقت، بازگشت و خروج نهایی هر کاربر در تقویم
- نمایش کاربران غایب هر روز
- نمایش جمع پاداش و اضافه‌کاری ماه
- دکمه «ورود و خروج» داخل شناسنامه کاربر
- تقویم اختصاصی ورود و خروج هر کاربر داخل شناسنامه

## اجرا

```powershell
cd C:\dev\roller-workshop
pnpm install
pnpm --dir apps/api exec prisma db push
pnpm dev:api
pnpm dev:miniapp
pnpm dev:admin
```

آدرس‌ها:

- پنل کارگر: http://localhost:5173
- پنل ادمین: http://localhost:5174
- API: http://localhost:3000/api/health

## تغییر نسخه Sidebar Navigation UX

در این نسخه منوهای اصلی پنل ادمین از بالای صفحه به سایدبار منتقل شده‌اند و زیرمنوهای هر بخش، مثل «فعالیت‌های جاری / فعالیت‌های بسته‌شده» و «سفارش‌های باز / سفارش‌های بسته‌شده»، در بالای محتوای همان صفحه نمایش داده می‌شوند.
